git config --global user.name - Встановлення вашого імені користувача в Git

git config --global user.email - Встановлення твоєї пошти

git init - Створіть порожній репозиторій Git або повторно ініціалізуйте наявний

git status - оказує поточний стан вашого робочого каталогу Git та області виконання.

git add - Додати вміст файлу до індексу

git commit - Запис змін до сховища

git-log - Показати журнали фіксації

git-checkout - Перемикання гілок або відновлення робочих файлів дерева

git-branch — перелік, створення або видалення гілок

git-push - Оновлення віддалених посилань разом із пов'язаними об'єктами

git-pull - отримати з іншого репозиторію або локальну гілку та інтегрувати з ним

git-merge - об'єднати дві або більше історій розробки разом

git-clone - клонувати сховище в новий каталог

gitignore – визначає навмисно невідстежувані файли, які потрібно ігнорувати

git-revert - Повернути деякі існуючі коміти

git-reset - Скидання поточного HEAD до вказаного стану

git-rebase - Повторне застосування фіксацій поверх іншої базової підказки

git-cherry-pick - Застосуйте зміни, внесені деякими існуючими комітами

git stash - зберігає зміни в брудному робочому каталозі
